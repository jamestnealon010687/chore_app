## Quick Setup (GitHub Pages â€” Free Hosting)

### 1. Create a new GitHub repository
- Go to [github.com/new](https://github.com/new)
- Name it something like `chore-champions`
- Set it to **Private** (so only you can see it)
- Click **Create repository**

### 2. Upload the files
- On your new repo page, click **"uploading an existing file"** link
- Drag and drop both `index.html` and `manifest.json` into the upload area
- Click **Commit changes**

### 3. Enable GitHub Pages
- Go to your repo's **Settings** tab
- In the left sidebar, click **Pages**
- Under "Source", select **Deploy from a branch**
- Under "Branch", select **main** and folder **/ (root)**
- Click **Save**
- Wait ~1 minute, then your site will be live at:
  `https://YOUR-USERNAME.github.io/chore-champions/`

### 4. Add to iPad Home Screen
- Open the URL in **Safari** on your iPad
- Tap the **Share** button (square with arrow)
- Tap **"Add to Home Screen"**
- Name it "Chore Champions" and tap **Add**
- It now works like a native app! ðŸŽ‰

## Features
- Side-by-side chore columns for 2 kids + optional guest
- Big emoji icons next to each chore (great for pre-readers)
- 6 different random celebration animations when tasks are completed
- Passcode-protected admin panel (default: `1234`)
- Add/remove/rename kids and chores
- Auto-resets every morning
- All data stored locally on the iPad

## Notes
- Works offline once loaded (all code is in a single HTML file)
- Data persists in the browser's localStorage
